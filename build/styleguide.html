Tooltip Widths

The default width of the tooltip is determined by the width of the original element and the text width. Utillize modifier classes to change the fixed width of the tooltip. For example .tooltip-sm sets the width to 104px, .tooltip-md sets the width to 136px, .tooltip-lg sets the width to 200px.

Spinners

The spinner levels are determined by their size. The base spinner is 100px+ and therfore moves slower. The small spinner is under 16px and so it moves faster and more dramatic. These sizes are defined here to help you in the creation of custom components. Spinner size and speed are already written into buttons and inputs.
